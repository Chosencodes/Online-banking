# Credix Vault Bank - Modern Banking Platform

## Overview
Credix Vault Bank is a modern, user-friendly banking platform built with Next.js, offering a comprehensive suite of banking features with a beautiful and intuitive interface. The platform supports both light and dark modes for optimal viewing in any environment.

## Features

### 1. Authentication
- Secure login system
- Token-based authentication
- Automatic token refresh
- Protected routes

### 2. Dashboard
- Account overview with total balance
- Quick action buttons for common tasks
- Recent transactions list
- Spending analytics with charts
  - Category-wise spending (Pie chart)
  - Monthly spending trends (Bar chart)

### 3. Accounts Management
- View all bank accounts
- Account details including:
  - Account type
  - Balance
  - Account number (last 4 digits)
- Quick actions per account:
  - Transfer funds
  - View details
- Add new account functionality

### 4. Transactions
- Detailed transaction history
- Transaction information includes:
  - Description
  - Category
  - Date
  - Amount
  - Transaction type
- Visual indicators for different transaction types

### 5. User Interface
- Responsive design for all devices
- Light/Dark mode support
- Modern, clean interface
- Intuitive navigation
- Real-time data updates

## How to Use

### 1. Getting Started
1. Visit the website at `www.greenroot.live`
2. Click "Login" to access your account
3. Enter your credentials

### 2. Dashboard Navigation
- Use the top navigation bar to switch between:
  - Dashboard
  - Accounts
  - Transactions
- Quick actions are available on the dashboard for common tasks

### 3. Managing Accounts
- View all accounts on the Accounts page
- Click "View Details" to see full account information
- Use "Transfer" to move funds between accounts
- Click "Add New Account" to create a new account

### 4. Viewing Transactions
- Access the Transactions page to see your history
- Transactions are sorted by date (newest first)
- Use the filters to find specific transactions
- Click on a transaction for detailed information

### 5. Customizing Your Experience
- Toggle between light and dark mode using the theme button in the navigation bar
- Choose between:
  - Light mode
  - Dark mode
  - System preference
